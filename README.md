# NutriSphere

NutriSphere is a full-stack AI-powered wellness and fitness tracking web application built using Next.js.

## Overview
NutriSphere helps users track and improve their daily health habits including sleep, workouts, hydration, calories, steps, BMI, and overall wellness score. It also includes an AI Wellness Coach for personalized guidance.

## Features
- AI Wellness Coach
- Sleep Tracking (bedtime, wake time, duration calculation)
- Workout Logging with calorie tracking
- Water Intake Tracker
- Step Counter
- BMI Calculator
- Wellness Score system
- Progress Analytics dashboard
- Wellness Courses section
- User authentication system

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Node.js (backend services)

## Project Status
This project is in active development and deployed for demonstration purposes.
